# powerbi-analise-retalho
Dashboard de Análise de Retalho (Power BI)
- Dashboard analítico para avaliar produtos próximos da data de validade no retalho. Inclui quatro páginas — Visão Geral, Lucro, Prejuízo e Influências nas Vendas — com métricas DAX, análise de vendas e descontos, identificação de marcas e SKUs mais rentáveis, impacto do prazo de validade no comportamento de compra e comparação entre tipos de loja.
